# gitnow
> Easy and fast git pull, push and more in [Fish Shell](https://fishshell.com/).

#### Commands:

- `pull` -> `git pull --rebase` and `git stash`
- `push` ->  `git push --set-upstream`
- `commit` -> `git add -A` and `git commit .`
- `upstream` -> `commit` and `push` commands.
